# PROFIT · Plugin de Presupuestación de Proyectos

PROFIT ayuda a generar la plantilla de importación para el sistema de presupuestación de proyectos geotécnicos.

## Skills disponibles

- **definicion** — guía al usuario bloque a bloque para recoger todos los datos del proyecto: identificación, personal, equipos, materiales, subcontratas, seguros y costes; al terminar genera el archivo de importación listo para cargar en Profit
- **planificacion** — recoge el planning de la obra preguntando por las fases, fechas y hitos del proyecto

## Cómo usar

1. Instala el plugin en Claude
2. Lanza **definicion** para iniciar la recogida de datos de un nuevo proyecto
3. Sigue el formulario guiado bloque a bloque
4. Al confirmar, se genera el archivo de importación listo para Profit
